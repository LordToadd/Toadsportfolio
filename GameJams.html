Portfolio Commissions About Me

Wren

Wren was my first game jam with my teammate Rowan Dykstra and the game that we won first place with. I did all the artwork for the game and Rowan did all the programming. This game is a 3d puzzel game where the player must find their way to the top of the tower using a grappling hook and a special gravity system. If you would like to play the game you can find it here: Wren

Rowvolver

Rowvolver was my second game Jam with Rowan Dykstra. I did most of the artwork for this game was well and Rowan did the programming. This is a game where we combined real world images with my 3d models and art style to make a unique look. The goal of the game is to get past as many floors as you can by defeating all the enamies on every floor. If you would like to play Rowvolver you can find it here: Rowvolver

Moe T Cycle

Moe T Cycle was my third game jam with Rowan although we did not end up finishing the game. It was a rhythm game where you play as Moe and you have to steer on beat to the music to not crash.
